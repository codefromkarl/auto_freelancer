# BidMaster 文档索引

当前系统以 **Python Manual Pipeline** 为主，不再依赖 n8n 自动编排。

## 目录结构

- `CORE_PRINCIPLES.md`：项目核心准则（强制）
- `getting-started/`：入门文档
- `architecture/`：架构与 API 文档
- `workflows/MANUAL_WORKFLOW.md`：手动流水线执行说明
- `guides/USER_GUIDE.md`：日常操作指南
- `scripts/manual_pipeline/README.md`：脚本层执行说明

## 快速导航
